# SPOTIFY CLONE 

Created Spotify Clone using:
Frontend Framework: **React JS** <br>
Frontend Library: **Tailwind CSS** <br>

Create a Vite + React App using: <br>
``npm create vite@latest ./``

Install Node Modules:

``npm install``

Local Host run command:

``npm run dev``
